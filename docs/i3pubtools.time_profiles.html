i3pubtools.time_profiles.generic_profile module¶

class i3pubtools.time_profiles.generic_profile.GenericProfile¶

Bases: object

A generic base class to standardize the methods for the time profiles. While I’m only currently using scipy-based probability distributions, you can write your own if you want. Just be sure to define these methods and ensure that the PDF is normalized!
